# Diretório de Imagens

Este diretório contém os assets estáticos da aplicação.

## Logo VSA

Adicione o arquivo da logo VSA Soluções em Tecnologia aqui com o nome:
- `vsa-logo.png` (recomendado)
- Ou `vsa-logo.jpg` / `vsa-logo.jpeg`

O componente Logo.tsx procurará por `/images/vsa-logo.png` por padrão.
